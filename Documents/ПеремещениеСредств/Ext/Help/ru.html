Возможные операции, выполняемые документом перемещение, коротко:

1. Перемещение средств между бюджетируемыми кошельками

2. Перемещение средств между кошельками вне бюджета

3. Перемещение вида: расход из бюджетного кошелька, приход во внебюджетный кошелек

4. Перемещение вида: расход из внебюджетного кошелька, приход в бюджетный кошелек

Теперь подробно.

1. Перемещение средств внутри бюджета. В документе не надо указывать статью ДР, так как остаток бюджета не меняется.

2. Аналогично, внебюджетное перемещение не меняет остаток по бюджету, статью ДР не выбираем.

3. Расход по бюджету. Выбирается расходная статья ДР. Остаток бюджета уменьшается. Если источник средств - кредитная карта, то автоматически будет сделано движение по бюджету по статье "Взятие кредита Овердрафт", конечно, если в кошельке эта статья указана в соответствующем реквизите.

4. Приход по бюджету. Выбирается приходная статья ДР. Остаток бюджета увеличивается.

Нюансы.

Снятие наличных с кредитной карты.
Источник - кредитка (внебюджетный), приемник - наличные. Автоматом сформируется поступление овердрафта по бюджету, т.е. статью ДР "Взятие кредита Овердрафт" выбирать не надо, чтобы не сформировалось еще одно такое же движение по бюджету. Однако можно и выбрать, для наглядности. В этом случае сформируются 3 движения по БДР:
1) автоматическое взятие овердрафта
2) явное взятие овердрафта по выбранной в документе статье, если та же статья указана в кошельке кредитки для взятия овердрафта
3) сторнирование предыдущей проводки
так сделано, чтобы не усложнять модуль проведения документа Перемещение

Гашение овердрафта.
Источник - любой бюджетный кошелек, приемник - кредитка (внебюджетный). Статья ДР - расходная, для возврата полученных кредитов.

Закрытие депозита делается двумя документами:
1. первый перемещает сумму сбережения (сумма открытия депо плюс сумма всех пополнений)
2. второй возвращает или перемещает сумму процентов (только сумма процентов)
все это актуально, если вы заинтересованы в раздельном учете процетов, иначе можно делать закрытие депо одним перемещением.
для раздельного учета процентов и основной суммы депозита, нужно начисление процентов отражать по сбережениям по отдельной статье.
это если начисление процентов не бюджетируется (если суммы процентов маленькие, 1-10-100 рублей, а депозитов несколько, то из-за округлений может сильно расходиться план и факт бюджета, поэтому проще проценты не бюджетировать, а проводить по бюджету в момент закрытия депозита, вторым перемещением)
