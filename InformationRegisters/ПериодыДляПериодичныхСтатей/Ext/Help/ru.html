Данный регистр используется совместно с обработкой "Заполнение месяца по шаблону".

В этой обработке выполняется создание поступлений и расходов средств на основании шаблона периодичности.

т.е. если у нас есть какие-то периодические платежи, то удобнее их вводить на следующий месяц массово, нажатием одной кнопки, чем копировать документ из прошлого периода и менять дату.

 

Логика работы описана ниже.

 

Для периодичности "Год".

 

Для периодичности "Квартал".

 

 

Для периодичности "Месяц".

 

 

Для периодичности "Неделя".

 

 

Для периодичности "Рабочий день".

 

 

Для периодичности "День".
