Содержит информацию об учетных записях электронной почты, которые используются для получения и отправки сообщений по электронной почте.

Для учетных записей, получение и отправка с которых выполняется автоматически без участия пользователя, должен быть обязательно введен пароль. Также для корректной работы конфигурации обязательно должна быть настроена системную учетную запись электронной почты.
